# Almaas Labbai Portfolio

> AI & Data Science Engineer | ML Engineer | Data Analyst
A modern and elegant personal portfolio website built from scratch using React + Vite, featuring a dark maroon and rose gold design system, smooth animations, scroll reveal effects, and a role-specific resume download modal.


This portfolio was designed and developed entirely from scratch — no UI libraries, no templates.
Every component, animation, and interaction was hand-crafted using plain React and CSS.

The portfolio highlights my experience, technical skills, internships, certifications, and real-world projects while providing recruiters with quick access to my resume and GitHub.

The goal was to build something that stands out visually while remaining professional and recruiter-friendly — clean typography, structured content, and real project metrics.


## Features

- Animated Hero Section — Typewriter role animation, glowing profile image with orbiting rings, and floating particle effects
- Stats Bar — Quick snapshot: internships, projects, technologies, availability
- About Me — Clear, professional introduction with highlights
- Projects Section — 5 projects with real metrics (accuracy, dataset size, role), numbered layout, accent hover effects
- Skills Grid — Categorized technology pills (no progress bars) — Languages, Libraries, ML & AI, Visualization, Databases, Tools
- Experience Timeline — 3 internships with company, role, period and description
- Certifications — Deloitte Data Analytics + B.E. degree
- Resume Modal — Role-specific resume download popup (ML Engineer, Data Analyst, Python Developer)
- Contact Section — Direct Gmail compose link, LinkedIn, GitHub, location
- Scroll Reveal Animations — Sections fade in on scroll
- Fully Responsive — Mobile, tablet, desktop

## Getting Started

### Prerequisites
- Node.js v18+
- npm

### Install & Run

```bash
npm install
npm run dev
```


## Deployment

This project is configured for deployment on **Vercel**.

To deploy your own version:
1. Fork this repository
2. Go to https://vercel.com → Import Project
3. Select this repository → Framework: **Vite**
4. Click **Deploy**
